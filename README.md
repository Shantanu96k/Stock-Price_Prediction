# Stock Price Prediction System

A web-based **Stock Price Prediction System** that uses **Machine Learning models** to forecast future stock prices.
The application allows users to **upload stock datasets, train models, and visualize predictions through interactive charts**.

This project demonstrates practical implementation of **Data Analysis, Machine Learning, and Web Deployment using Flask**.

---

## 🚀 Features

* 🔐 **User Authentication**

  * User Registration
  * Secure Login System
  * MySQL database integration

* 📂 **Dataset Upload**

  * Upload CSV stock datasets
  * Automatic preprocessing

* 🤖 **Multiple ML Models**

  * LSTM (Deep Learning)

* 📊 **Interactive Visualization**

  * Plotly interactive graphs
  * Historical vs Predicted price comparison
  * Future price forecasting (next 30 days)

* 📥 **Export Results**

  * Download predictions as CSV
  * Visual dashboard

---

## 🧠 Machine Learning Workflow
